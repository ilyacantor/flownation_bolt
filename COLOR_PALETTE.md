# FlowNation Color Palette

## Brand Colors

### Primary - Cyan
- **Cyan 400**: `#00c6d6` / `rgb(0, 198, 214)` - Main brand accent, headings, highlights
- **Cyan 500**: `#0dcaf0` - Interactive elements, buttons
- **Cyan 600**: `#0bb5c9` - Button hover states
- **Cyan 700**: `#099faf` - Active states
- **Cyan 900**: `rgba(6, 182, 212, 0.2)` - Subtle backgrounds, gradients

### Secondary - Orange
- **Orange 400**: `#ff8c00` / `rgb(255, 140, 0)` - Coach persona, secondary CTA
- **Orange 500**: `#ff7a1a` - Links, accents
- **Orange 600**: Hover states for orange elements

### Accent Colors

#### Green
- **Green 400**: `#22c55e` - Team persona, success states
- **Green 500**: `#16a34a` - Active/success indicators

#### Blue
- **Blue 400**: `#3b82f6` - Shop persona, informational elements
- **Blue 500**: Alternate interactive states

## Neutral Colors

### Dark Backgrounds
- **Primary Background**: `#0E1013` - Main app background
- **Secondary Background**: `#0c0f14` - Content sections
- **Card Background**: `#1a1d23` - Card base
- **Card Gradient**: `linear-gradient(145deg, #1a1d23, #252930)` - Elevated cards
- **Hover Background**: `linear-gradient(145deg, #1f2229, #2a2f38)` - Interactive card states

### Grays
- **Gray 900**: `#111827` - Dark surfaces, overlays (with opacity: `#111827/80`, `#111827/90`)
- **Gray 800**: Border colors for cards
- **Gray 500**: `#6b7280` - Muted text, labels
- **Gray 400**: `#9ca3af` - Secondary text
- **Gray 300**: `#d1d5db` - Primary body text on dark backgrounds

### White with Opacity
- **White**: `#ffffff` - Primary text
- **White/90**: `rgba(255, 255, 255, 0.9)` - Icons, emphasized text
- **White/10**: `rgba(255, 255, 255, 0.1)` - Subtle borders
- **White/8**: `rgba(255, 255, 255, 0.08)` - Very subtle borders
- **White/5**: `rgba(255, 255, 255, 0.05)` - Card backgrounds, subtle surfaces

## Effects & States

### Shadows
- **Card Shadow**: `0 4px 12px rgba(0, 0, 0, 0.3)`
- **Elevated Shadow**: `0 8px 24px rgba(255, 140, 0, 0.3)` (orange glow)
- **Button Shadow**: `shadow-lg` - Standard Tailwind large shadow

### Glows
- **Cyan Glow**: `0 0 8px rgba(0, 198, 214, 0.4)`
- **Orange Glow (Pulse)**: `0 0 8px rgba(255, 140, 0, 0.6)` to `0 0 16px rgba(255, 140, 0, 0.9)`

### Borders
- **Default**: `border-white/10` or `border-gray-800`
- **Hover - Cyan**: `border-cyan-500/50` or `border-cyan-400/50`
- **Hover - Orange**: `border-orange-500/50`
- **Hover - Green**: `border-green-500/50`
- **Hover - Blue**: `border-blue-500/50`

### Gradients

- **Background Gradient**: `from-cyan-900/20 to-[#0E1013]`
- **Card Gradient**: `linear-gradient(145deg, #1a1d23, #252930)`

## Typography

### Font Family
- **Primary**: `'Quicksand', sans-serif` - All text elements

### Text Colors
- **Primary Heading**: `text-white`
- **Secondary Heading**: `text-cyan-400`
- **Body Text**: `text-gray-300`
- **Muted Text**: `text-gray-400` or `text-gray-500`
- **Link Hover**: `hover:text-cyan-400`

## Animation Colors
- **Breathe/Pulse**: Orange glow effects
- **Hover Transitions**: Cyan/Orange border and shadow changes
